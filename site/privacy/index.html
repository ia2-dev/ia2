IA² Navigator browser extension

Privacy, in plain terms.

IA² Navigator inspects supported pages on your device. It does not send page content, browsing history, or usage analytics to IA² or Inferal.

Last updated 17 July 2026

What the extension processes

IA² Navigator reads the top-level content and URL of the current page to extract HTML/RDF statements and present their graphs, terms, and source carriers. On a HARE resource envelope, it also reads the manifest and its embedded representations so it can verify and present the declared files.

The extension checks eligible pages locally for a HARE declaration so that it can open the file view automatically. On other pages, the full Navigator opens only when you select its toolbar action.

What we collect

Inferal and the IA² project do not collect or receive the page content, URLs, RDF statements, HARE resources, browsing history, searches, or interaction data processed by the extension. The extension has no analytics, advertising, account system, or developer-operated data service.

Processing occurs in the page and extension contexts in your browser. Parsed statements, verified HARE resources, and fetched previews remain in memory and are discarded when their page context ends.

Local storage

The Navigator stores only its drawer position, size, and docking preference in the current website's browser session storage. It does not store extracted page content. Your browser clears this session state according to its normal session-storage behavior.

Requests you initiate

When you choose to retrieve a linked RDF source or open a remote definition preview, the extension requests that resource directly from its published address. Retrieval requests made by the Navigator omit browser credentials and the referring page. Some definition previews can load the destination in a sandboxed browser frame.

The destination service can receive ordinary request information, such as your IP address and the requested URL, and is governed by its own privacy policy. The request is not routed through IA² or Inferal, and the response is not sent to us.

Sharing and human access

We do not sell, share, or use extension data for advertising, profiling, credit decisions, or any purpose unrelated to the extension's visible inspection and file-browsing features. No person at IA² or Inferal can read content processed locally by the extension because that content is not sent to us.

Chrome Web Store Limited Use

IA² Navigator's use of information received from Chrome APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements. The extension uses access to page content only to provide its disclosed, user-facing purpose.

Changes and contact

If a release changes how the extension handles data, this page and the Chrome Web Store disclosures will be updated before that release is published.

Questions and privacy requests can be filed in the IA² issue tracker. IA² Navigator is open source, and its implementation is available in the browser extension package.
